## About

Indus Brain AI helps industries quickly access equipment details, manuals, SOPs, safety information, maintenance records, and worker reports using AI.

## Key Features

- 🤖 AI assistant for industrial questions
- 📄 Document and PDF intelligence
- 🏭 Equipment management
- 📝 Worker reports
- 🚨 Safety and emergency information
- 📊 Manager dashboard and analytics

## Technologies

`Google AI Studio` · `OpenRouter API` · `GitHub` · `Google Cloud`
